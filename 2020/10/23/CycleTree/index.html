基环树的一般处理方法

首先我们可以找到基环树上的环，然后采取一些方法破环成链[1]，然后就可以得到一个一般的树或森林，便可以进行一般的处理了。所以，关键的问题还是如何找环。

找环的方法

有向的基环树

long long FindCycle(int now){
    // 找环开始
	vis[now] = true;
	root = now;
	while(!vis[fa[root]]){
		root = fa[root];
		vis[root] = true;
	}
    // 找环结束
	DFS(root);
	long long tmpAns = std::max(f[root][0], f[root][1]);
	vis[root] = true;
	root = fa[root];
	DFS(root);
	return std::max(tmpAns, std::max(f[root][0], f[root][1]));
}

无向的基环树：

bool FindCycle(int now, int fa){
	vis[now] = true;
	for(int i = hd[now]; i; i = e[i].next){
		if(e[i].v == fa){
			continue;
		}
		if(vis[e[i].v]){
			rootA = now, rootB = e[i].v;
			return true;
		}
		else{
			if(FindCycle(e[i].v, now)){
				return true;
			}
		}
	}
	return false;
}
